Git Cheat Sheet

Commandes Description
git init Initia­lis­ation d'un répertoire
git status Affiche le statut
git add Ajoute un fichier
git add * Ajoute les nouveaux fichiers
git delete Supprime un fichier
git commit -m "the messag­e" Créer un commit avec un message
git log Affiche les commits antérieurs à l'actif
git show Affiche les infos d'un commit
git branch Crée une branche
git branch -v Affiche les branches courantes
git checkout Switch sur une branche ou un commit
git checkout -b Créer une nouvelle branche et switch dessus
git merge Fusionne la branche citée et le branche active
git remote add
git pull
git clone Clone un projet sur sa machine
git help Détails d'une commande
git help Liste des commandes
